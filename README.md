# 🛒 Retail Sales Analysis System

## 📌 Project Overview

The Retail Sales Analysis System is a Python, SQL, and MySQL-based data analysis project designed to analyze retail sales data and perform data quality validation. The project uses Pandas and NumPy to clean, process, and analyze data while generating valuable business insights.

This project demonstrates practical skills in SQL querying, data cleaning, data validation, business analysis, and CSV export.

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/yuvrajnishad638717-eng/Retail-Sales-Analysis-System.git
```

### 2. Install dependencies

```bash
pip install pandas numpy mysql-connector-python
```

### 3. Create the MySQL database and import the required tables.

### 4. Update your MySQL connection details

```python
conn = mysql.connector.connect(
    host="localhost",
    user="root",
    password="YOUR_PASSWORD",
    database="retail_sales"
)
```

### 5. Run the project

```bash
python Retails_sales.py
```
